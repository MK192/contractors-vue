# contractors-vue

## Project Description
Application for registration finding and message contractors.
Loged user can register as contractor after signing form, after that he can receive message/offers.
Authenticated contractors can read messages or edit profile.

### Instructions
- Clone repository.
- Install dependency with npm install or yarn install.
  Start project with npm run serve.

### Technologies
- Vue
- Vuex
- Vue-router
- Tanstack query
