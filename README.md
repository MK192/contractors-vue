# contractors-vue

## Project Description
Application for registration finding and message contractors.
Loged user can register as contractor after signing form, after that he can receive message/offers.
Authenticated contractors can read messages or edit profile.

### Instructions
- Clone repository.
- Install dependency with npm install or yarn install.
- Create project with firebase console.
- Add .env.local file outside src folder in the root of app.
- Inside .env.local file add VUE_APP_WEB_API_KEY="Your Web APY key".
- You can find Web APY key in Project Settings inside your project firebase dashboard.
- Start project with npm run serve.

### Technologies
- Vue
- Vuex
- Vue-router
- Tanstack query
